#### Styling components.

Styling components is done using `style props` and is as easy as using `HTML` attributes.

Add a `background-color` and `color` to a `Button`.

```jsx
<Button bg="tomato" color="#fff">
  Button
</Button>
```

<Block src="callout">

We have added support for every CSS properties such as `fontSize`, `fontFamily`, `textDecoration`, `flexDirection` and much more.

</Block>

#### A hero

```jsx
<Section py="20px" textAlign="center" bg="#f7f7f7">
  <H1 my="0">Hero Heading</H1>
  <P fontSize="20px">This is some text</P>
  <Button bg="#08f" color="#fff" borderWidth="0" hoverBg="#000">
    Get Started
  </Button>
</Section>
```
