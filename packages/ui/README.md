# @reflexjs/ui

A tiny library for creating and styling UI components with [Theme UI](https://theme-ui.com).

## Installation

```jsx
npm i @reflexjs/ui theme-ui
```

## Example

```jsx
import { Button } from "@reflexjs/ui"

export default () => (
  <Button
    bg="primary"
    color="text"
    px="2"
    py="1"
    fontSize="sm"
    hoverBg="secondary"
    hoverColor="white"
  >
    Button
  </Button>
)
```
