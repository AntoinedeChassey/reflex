## What is Reflex?

A collection of **plugins**, **themes** and pre-built **blocks** to help you build Gatsby sites faster.

## Requirements

Make sure you have the Gatsby CLI installed.

```sh
npm i -g gatsby-cli
```

## Create a site

The easiest way to use Reflex is to use the `base` starter.

**Step 1: Create site**

To create a new site using the `base` starter run the following command:

```sh
gatsby new NAME_OF_SITE reflexjs/gatsby-starter-base
```

**Step 2: Configure Gatsby**

Edit `gatsby-config.js` file and configure your site name and description.

```js title=gatsby-config.js
require(`dotenv`).config()

module.exports = {
  siteMetadata: {
    // highlight-start
    title: "Reflex",
    description: "Starter for Reflex.",
    // highlight-end
    siteUrl: process.env.SITE_URL || "http://localhost:8000",
  },
  plugins: ["@reflexjs/gatsby-theme-base"],
}
```

**Step 3: Run your site.**

You can now run your Gatsby site:

```sh
gatsby develop
```

**Step 4: View your site**

Gatsby will start your development environment at [http://localhost:8000](http://localhost:8000).
