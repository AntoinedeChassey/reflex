<P fontSize="xl|2xl" mt="2" mb="0">
  The base starter includes a set of pre-defined design tokens that you can use
  to style your components.
</P>

Use [style props](/docs/theming#style-props) to theme your components.

```jsx
<P fontSize="md" />
```

For responsive styles, separate your values using a pipe: `|`.

```jsx
<P fontSize="sm|md|lg|2xl" />
```

<Block src="callout" rounded="lg">

**Note:** Any props not listed below such as `textDecoration` or `transition` or `opacity`) can use default CSS values.

```jsx
<P textShadow="underline" />
```

</Block>
